Check Availability

Enter your desired subdomain name in the search bar to see if it's available.

3

Configure DNS

Point your subdomain to your server or hosting provider (Vercel, Netlify, etc.).

Frequently Asked Questions

Yes! is-app.top is completely free for everyone. We believe in supporting the developer community by providing accessible tools for projects of all sizes.
Registration is done via GitHub Pull Request. Simply fork our repository, add a JSON file with your desired subdomain name in the domains/ directory, and submit a PR. Once merged, your domain is active!
We currently support CNAME, A, AAAA, MX, and TXT records. This allows you to point your subdomain to any hosting provider, server IP, or even set up email verification.
If you have any questions or need assistance, you can open an issue on our GitHub repository or email us directly at contact@is-app.top.
